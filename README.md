# Tooltip UI

Create a tooltip for navigation items using only HTML and CSS.

## Features

- Tooltip appears above navigation items on hover.
- Fade-in and slide-up animation.
- Pure HTML and CSS, no JavaScript.

## Files

- `index.html` - HTML structure
- `style.css` - CSS styling

## Preview

![Tooltip Preview](tooltip-preview.png) *(if you have an image)*

## How to Use

1. Open `index.html` in a web browser.
2. Hover over the navigation items to see the tooltips.

## Customization

- Adjust the tooltip text in the HTML.
- Modify colors, fonts, and animations in `style.css`.
